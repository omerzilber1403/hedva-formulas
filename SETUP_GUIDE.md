# 🧮 Calculus 2 Formulas Pro - Setup Guide

## ✅ המצב הנוכחי
הפרויקט שלך מוכן ופועל! הרפקטורינג הושלם בהצלחה.

---

## 📋 מה עשינו?

### 1️⃣ **קבצים שעברו שדרוג מלא:**
- ✅ **index.html** - אפליקציה מלאה עם React + KaTeX + אימות משתמשים
- ✅ **quiz_data.js** - 60 נוסחאות חדו"א 2 (במקום שאלות קיימות)
- ✅ **netlify/functions/users.mjs** - Backend משודרג עם אימות PIN
- ✅ **netlify.toml** - תצורת Netlify (נוסף)

### 2️⃣ **תכונות חדשות:**
- 🔢 **KaTeX Integration** - רינדור מתמטי מושלם לנוסחאות LaTeX
- 🔐 **Authentication** - מערכת התחברות עם שם משתמש + PIN
- 💾 **Cloud Sync** - שמירת התקדמות ב-Netlify Blobs
- 🎴 **Flashcards Mode** - כרטיסייה: חזית (נושא+שם), גב (נוסחה)
- 📝 **Test Mode** - מבחן עם 4 תשובות מתמטיות משולבות
- 🎨 **Purple Theme** - עיצוב מדעי בגווני סגול/אינדיגו/ויולט
- 📊 **High Score Tracking** - מעקב אחר ציון הגבוה ביותר
- 🔢 **Quiz Counter** - ספירת מבחנים שהושלמו

## 🧪 איך לבדוק את האפליקציה?

### 1️⃣ **פתח דפדפן:**
```
http://localhost:8888
```

### 2️⃣ **צור משתמש חדש:**
- לחץ על "צור חשבון חדש"
- הזן שם משתמש (לדוגמה: "Student1")
- הזן PIN (4 ספרות, לדוגמה: "1234")
- לחץ "צור חשבון"

### 3️⃣ **בדוק את מצבי הלימוד:**

#### **📚 Flashcards Mode:**
- לחץ על "כרטיסיות"
- בצד הקדמי: נושא + שם הנוסחה (בעברית)
- לחץ על הכרטיס להפוך
- בצד האחורי: נוסחה מתמטית מרונדרת ב-KaTeX
- השתמש בכפתורים: ❌ (אדום), ⚠️ (צהוב), ✅ (ירוק)

#### **✏️ Test Mode:**
- לחץ על "מבחן"
- כל שאלה מציגה נוסחה + 4 תשובות (שם הנוסחה)
- כל התשובות מרונדרות עם KaTeX
- עקוב אחר הציון בזמן אמת
- בסיום: ראה ציון סופי + highScore

#### **📋 List Mode:**
- לחץ על "רשימה"
- ראה את כל 60 הנוסחאות במבט אחד
- סנן לפי סטטוס (אדום/צהוב/ירוק/לא מסומן)
- כל נוסחה מוצגת עם KaTeX

### 4️⃣ **בדוק Cloud Sync:**
- לחץ על "Sync with Cloud"
- אם הגדרת Netlify Blobs נכון, הנתונים יישמרו
- התנתק והתחבר שוב - הנתונים שלך יישמרו!

## 🔧 פתרון בעיות נפוצות

### 1. **השרת לא עולה:**
```powershell
# עצור את השרת הקיים (Ctrl+C)
# התחל מחדש:
netlify dev
```

### 2. **KaTeX לא מציג נוסחאות:**
- וודא שיש אינטרנט (KaTeX נטען מ-CDN)
- פתח את Developer Tools (F12) וחפש שגיאות בקונסולה

### 3. **Cloud Sync לא עובד:**
- זה נורמלי בסביבת פיתוח מקומית
- כדי להפעיל Netlify Blobs:
  1. צור חשבון ב-Netlify.com
  2. קשר את הפרויקט: `netlify link`
  3. הגדר environment variables אם נדרש

### 4. **אין נתונים:**
- וודא ש-quiz_data.js נטען נכון
- פתח F12 → Console → הקלד `window.__QUIZ_DATA__`
- צריך לראות מערך של 60 אובייקטים

## 🎓 שימוש במערכת

### **למידה עם Flashcards:**
1. התחל עם "כרטיסיות"
2. למד כל נוסחה: נושא → הפוך → נוסחה
3. סמן ירוק ✅ לנוסחאות שאתה יודע
4. סמן צהוב ⚠️ לנוסחאות שאתה בטווח
5. סמן אדום ❌ לנוסחאות שאתה צריך לחזור עליהן

### **אימון עם מבחנים:**
1. לך ל"מבחן"
2. 10 שאלות אקראיות
3. לכל שאלה - בחר את השם הנכון של הנוסחה
4. בסיום - ראה ציון + highScore

### **סקירה מלאה:**
- השתמש ב"רשימה" כדי לראות את כל הנוסחאות
- סנן לפי סטטוס כדי למקד את הלימוד
